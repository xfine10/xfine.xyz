D17:鲁朗—林芝

05-18-2014 3 min read # 2014 # 西藏 骑行

⛅：☀ 📆：2014.5.18
　　昨晚没来得及看瑞士的风光，今天出发就好好赏心悦目吧，一路上坡，景色优美，还真有那么点瑞士的味道，可惜今天路上太多的解放军车队，弄得路都被他们占了。爬坡真心累！刘艳那孩子还打电话给我，我各种的打击，她还是忍着关心我，问候我，我知道她放心不下我，也放不下我，边说边推，满眼望去，四周全是雪山，林海。 慢慢爬21公里大上坡到色季拉山垭口，今年第一次吃泡面，吃了东西拍了照，等达哥和老爷子好久他们才来。今天运气不好，没有看到南伽巴瓦峰，世界第十六高峰，又叫羞女峰，很难见真面目。 还在色季拉山脚下庆幸自己此行运气好：见了别人见不到的梅里日出，没有摔过车、扎过胎，断过链……看看其他人，呵呵……妈的，心里还没乐呵完，才到4177发现我那烂车真的烂了，尼玛山地变死飞了，哥哭了[/大哭]，爬到山顶队友来自珠海65岁的曲老说：坏了，修吧！那会砸车的冲动都有。
　　下山哥们可是拼命的踩，压刹车，话说可是死飞啊，得了，哥们一边看徒步的搭车过去心里刚鄙视完，一边低头看我那死飞的链条不踩会怎么，刚抬头，尼玛急转弯道，哥都快撞上护栏了，一把龙头内拧，尼玛路基边上全是沙石，不用说摔吧，摔，真摔了，漂亮的摔了，哥们爬起来活动活动筋骨，哈哈，哥没事，那可是43码摔车，既然皮都没有掉，第一时间把车给踢了几脚，扶起来再摔几次，尼玛烂车，害死我了，啷哩个啷，人没事。正准备拍照一对解放军车队一字排开上来，赶快扶起烂车走人，哥们可丢不起那人。一路提心吊胆骑到林芝，路上遇到个妹子兰州骑过来的妹子（骑过来的人家不搭车）挺厉害的哈，走的时候把我超了，哭吧，你妹，什么世道？谁叫我车烂了呢？快到林芝还在路边砍了西瓜等他们，味道不错，卖瓜那哥们一副文青样，一聊还真是文青，可惜是个卖西瓜的文青。在路上还遇到发传单的车子硬给我塞广告让我住他家客栈，还说他家现在只有女儿在家，我听着怎么有点其他意思……
　　到林芝直奔美利达，老板一看马上修好，吓死我了烂车，还以为不让我到拉萨的节奏……尼玛倒霉的一天[/糗大了]晚上还吃了屎一样难吃的饭，林芝是这一路除香格里拉外第二个有红绿灯的城市，气候很好，鸟语花香，不愧是高原粮仓。修车，吃饭，找住宿，那个老板很客气的把我们带到他家宾馆住下，老爷子还让我买了鸡蛋回来用宾馆的烧水壶煮，还买了明天的早点，干粮，明天要早起出发不想因为吃早点浪费时间，可是第二天还是很晚出发了，我的睡眠啊！
